Application Key features

  • Backend app to allow admin user to perform CRUD book operations (add/remove/view/update books).
  • Frontend app use Spring Database Authentication and integrated with Java Mail API to allow mail generation for registration and order flows.
  • Frontend app also allows users to perform CRUD operations on Shipping/Billing entities.
  • Frontend app enables user to add books to shoppingcart, save cartitems and checkout order with order confirmation mail received.
