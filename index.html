Cabin Booking App

Wild Oasis is a booking management system for the staff worked at this resort, to help them manage customer, cabins and bookings overall. This app utlized React library with Supabase server API to easily manage app data. React Query is used to manage entity state to ensure performance is optmized with caching and prefetching tools. Rechart library is used to help staff visual business data with beautifully designed UI interface.

  • FrontEnd: React

  • Backend: Supabase Postgres database API

  • Third Party Libraries: React Query, React Routers, Recharts, React-Form and Styled Components

Online Bookstore App

Online Bookstore app provides both frontend portal for customers to make book purchase, and admin portal for bookstore staff to manage books. New Customer will be able to receive Mail Notification once gets registered and also received Mail Notification once order is placed. This bookstore permits all functionality with other ecommerce website such as add/delete/update books and orders.

  • FrontEnd: Thymeleaf Engine for UI

  • Backend: SpringBoot and Java

  • Other Tools: Java Mail API and Postgres Database

Blog RestFul API

This Blog Backend API allows users to manage blog post, post comments and post category securely with the use of JWT token for each of the http post and update request. Blog API contract is created via Swagger to allow all stakeholders understand this API easily.

  • Backend: Spring Boot Restful API

  • Other Tools: Postgres Database, JWT token and Swagger API documentation
